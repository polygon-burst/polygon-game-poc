WORK IN PROGRESS

HTML port of PoC Polygon Game (made with LibGDX), Input, GUI and Basic Game Mechanics test.

The Main controls are as follows:
- Mouse Left Button: Select, Increase
- Mouse Right Button: Decrease
- Keyboard Arrow Keys / Controller DPad, Analog Stick : Step, Increase/Decrease
- Keyboard W, S, Space, Enter Key / Controller A, X, R1 Button : Ok, Increase
- Keyboard A, D, Backspace Key / Controller B, Y, L1 Button : Cancel, Decrease
- Keyboard Escape Key / Controller Start, Back Button : Settings, Back

Game controls:
- Keyboard Arrow Keys / Controller DPad, Analog Stick : Movement
- Keyboard W, Space, Enter Key / Controller A, R1 Button : Increase Gravity Field
- Keyboard A, Backspace Key / Controller B, L1 Button : Decrease Gravity Field
- Keyboard S Key / Controller X Button : Toggle Gravity Field
- Keyboard D Key / Controller Y Button : Reset HPs
- Keyboard Escape Key / Controller Start, Back Button : Menu
