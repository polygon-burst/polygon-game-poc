WORK IN PROGRESS

Desktop and HTML port of PoC Polygon Game (made with LibGDX).

The controls are as follows:
- Arrow Keys / DPad / Analog Stick : Movement
- W & S & Space & Enter Key / A & X Button : Toggle Cloak
- A & D & Backspace Key / B & Y Button : Reset HPs
- Escape & Ctrl Key / Start & Back Button : Menu
