Desktop and HTML port of PoC Polygon Game (made with LibGDX).

The controls are as follows:
- Arrow Keys / DPad / Analog Stick : Movement
- W Key / Button A : Toggle Cloak
- A Key / Button B : Reset HPs
- Space & Esc Keys / Button Start & Back : Exit (not in HTML version)
