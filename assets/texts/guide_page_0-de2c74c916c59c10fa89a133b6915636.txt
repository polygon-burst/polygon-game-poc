The Main controls are as follows:

Mouse Left Button:
 - Select, Increase
 
Mouse Right Button:
 - Decrease
 
Keyboard Arrow Keys / Controller DPad, Analog Stick:
 - Step, Increase/Decrease
 
Keyboard W, S, Space, Enter Key / Controller A, X, R1 Button:
 - Ok, Increase
 
Keyboard A, D, Backspace Key / Controller B, Y, L1 Button:
 - Cancel, Decrease
 
Keyboard Escape Key / Controller Start, Back Button:
 - Settings, Back
